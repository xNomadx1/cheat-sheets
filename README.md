# Cheat Sheets

Small command reference sheets for common tools and workflows.

## Files

- `bash_commands.py` - Basic shell navigation, files, redirection, variables, shortcuts, and jobs.
- `linux_commands.py` - General Linux commands for files, processes, networking, services, storage, and packages.
- `git_commands.py` - Git setup, staging, branches, remotes, history, stash, recovery, and maintenance.
- `python_commands.py` - Python commands for running code, environments, packages, testing, formatting, and packaging.
- `rust_commands.py` - Rust commands for `cargo`, `rustup`, builds, tests, linting, docs, and publishing.

## Usage

Run any sheet with Python:

```bash
python3 git_commands.py
python3 python_commands.py
python3 rust_commands.py
```

Each file prints a grouped command list to the terminal.

## Notes

- **WARNING: A FEW COMMANDS ARE DESTRUCTIVE. READ THEM BEFORE RUNNING THEM.**
- Destructive commands are marked with a leading `*` before the command name.
- The sheets are meant to be quick references, not full documentation.
- Some commands assume the tool is already installed.
