# Cheat Sheets

Small command reference sheets for common tools and workflows.

## Files

- `bash_commands.txt` - Basic shell navigation, files, redirection, variables, shortcuts, and jobs.
- `cron_commands.txt` - Cron syntax, installing jobs, common schedules, logging, and troubleshooting.
- `linux_commands.txt` - General Linux commands for files, processes, networking, services, storage, and packages.
- `git_commands.txt` - Git setup, staging, branches, remotes, history, stash, recovery, and maintenance.
- `python_commands.txt` - Python commands for running code, environments, packages, testing, formatting, and packaging.
- `rust_commands.txt` - Rust commands for `cargo`, `rustup`, builds, tests, linting, docs, and publishing.
